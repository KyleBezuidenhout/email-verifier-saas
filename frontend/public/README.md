# Logo Placement

Place your `logo.png` file in this directory (`email-verifier-saas/frontend/public/`).

## Logo Specifications

- **File name**: `logo.png`
- **Recommended format**: PNG with transparent background
- **Recommended dimensions**: 180x40px (or maintain 4.5:1 aspect ratio)
- **File size**: Optimize for web (under 100KB recommended)

## Where the Logo Appears

The logo is automatically displayed in:

1. **Sidebar** (Dashboard navigation) - Height: 32px (h-8)
2. **Navbar** (Top navigation bar) - Height: 44px (h-11)  
3. **Footer** (Page footer) - Height: 40px (h-10)

All logo instances include:
- Hover opacity effect (90% on hover)
- Proper aspect ratio maintenance
- Responsive sizing
- Priority loading for performance

## Adding Your Logo

1. Remove the background from your logo (use tools like remove.bg, Photoshop, or GIMP)
2. Export as PNG with transparency
3. Save as `logo.png` in this directory
4. The logo will automatically appear across all components
